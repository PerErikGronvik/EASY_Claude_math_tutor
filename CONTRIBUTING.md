# Contributing

Thank you for your interest in contributing to Mathematical Concept Notebooks!

## How to Contribute

### 1. Report Issues or Missing Packages

**Found a missing dependency?**
- Open an [Issue on GitHub](../../issues)
- Or add it directly to `missing_packages.md` and submit a pull request

**Found a bug or have a suggestion?**
- Open an [Issue on GitHub](../../issues) describing the problem or feature request

### 2. Contribute Code or Notebooks

**To add new content or fix issues:**

1. **Fork this repository**
2. **Create a new branch** for your changes:
   ```bash
   git checkout -b my-contribution
   ```
3. **Make your changes:**
   - Add new notebooks
   - Fix bugs
   - Improve documentation
   - Add new course folders
4. **Test your changes** - ensure setup scripts work and notebooks run correctly
5. **Commit your changes:**
   ```bash
   git commit -m "Description of changes"
   ```
6. **Push to your fork:**
   ```bash
   git push origin my-contribution
   ```
7. **Open a Pull Request** on GitHub

### 3. Contact the Maintainer

For questions or discussions, open an Issue on GitHub or reach out to **PerErikGronvik** on GitHub.

## Guidelines

- Keep notebooks focused on one concept at a time
- Follow the Claude interaction guidelines in `.github/copilot-instructions.md`
- Test setup scripts before submitting
- Update `example.pyproject.toml` when adding new package categories
- Write clear commit messages

## Code of Conduct

Be respectful, helpful, and constructive. This project is for learning!
